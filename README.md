# MixtureLib

## Requirements
1. Python 3.6.2
2. pip 19.2.3

## Installation
1. git clone repository
2. cd MixtureLib
3. python3.6 -m pip install ./src/.

## Uninstallation
1. python3.6 -m pip uninstall MixtureLib

## Version
0.0.1
