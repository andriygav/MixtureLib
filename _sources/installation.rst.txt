************
Installation
************

Requirements
============

- Python 3.6.2
- pip 19.2.3

Installing on macOS
===================

Install
-------
.. code-block:: bash

	git clone https://github.com/andriygav/MixtureLib.git
	cd MixtureLib
	python3.6 -m pip install ./src/

Uninstall
---------
.. code-block:: bash

	python3.6 -m pip uninstall MixtureLib
